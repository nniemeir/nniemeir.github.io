## Niemeir Computing
My personal blog covering systems programming, system administration, FOSS programs, and cybersecurity

## Topics
- Low-level systems programming
- Deployment automation
- Security hardening and best practices

## Featured Articles
- [The Compaq Manifesto](https://nniemeir.github.io/blog/compaq_manifesto.html) - About my experience learning OS fundamentals at age 11
- [On Media Consumption & Logging](https://nniemeir.github.io/blog/media_logging.html) - My thoughts on logging media that we engage with through CSVs and discovering new media without relying on recommendation algorithms

## Projects
Writeups of the projects I have done that I believe to be of greatest educational value are on the site, including:
* [ClowniSH](https://nniemeir.github.io/projects/clownish.html) - POSIX-like shell implementation designed to annoy the user
* [Cyllenian](https://nniemeir.github.io/projects/cyllenian.html) - minimalist HTTPS server 
* [FilmFS](https://nniemeir.github.io/projects/filmfs.html) - FUSE filesystem that tracks video file viewings and logs them to a SQLite database
* [Halfway Across](https://nniemeir.github.io/projects/halfway-across.html) - parser-based interactive fiction game in Qt
* [Orlok](https://nniemeir.github.io/projects/orlok.html) - minimal strace-like process tracing utility for x86_64 Linux
